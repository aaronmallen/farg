# Farg

A comprehensive Rust library for colorimetry, color space conversions, and color manipulation.

Farg provides universal color space conversions, and perceptually accurate color manipulation. It's designed to serve
web developers with sensible defaults while giving colorimetrists full control.

## Installation

Add to your `Cargo.toml`:

```toml
[dependencies]
farg = "0.1"
```
